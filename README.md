## convert_eps.sh 

.png .jpg -> .eps にまとめて変換するスクリプトです.  
imageファイルにあるのでご自由にお使いください.

使い方
```
./convert_eps <dir_name>
```

convertコマンドがない方
```
# linux
sudo apt-get -y install imagemagick

# MacOS
brew install imagemagick
```
